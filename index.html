Git

  • git init - to initialize the git in local
  • git branch - to check all the branch
  • git checkout branch-name - to switch the branch
  • git checkout -b branch-name - to create new branch and switch to that branch
  • git add **file name** - to add the file in the git.
  • git commit -m "initial setup" - to add the commit/save message.
  • git status - to check all the files added/updated/deleted
  • git add . - to add all the files to staging area.
  • git stash - to stash the changes temporiarly in the current branch.
  • git stash apply - to add the stashed changes again to the current branch.
